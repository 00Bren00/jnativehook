*** About ***
JNativeHook was created to give java access to key and mouse events that are 
outside of the java sandbox.

*** Compiling ***

Unix/Linux:
	Dependencies
		Ant-1.7 or later
		Java 1.5 or later
		XOrg Server 1.8 or later
		XOrg Server development headers
		GCC 4.0 or later
		Make 3 or later
		
	Setup
		Use your package manager to install all dependencies.
		
		
Windows:
	Dependencies
		Ant-1.7 or later
		Java 1.5 or later
		XOrg Server 1.8 or later
		XOrg Server development headers
		MinGW - 
			GCC 4.0 or later
			Make 3 or later
		
	
		
Mac OS X:
	Dependencies
		Ant-1.7 or later
		Java 1.5 or later
		GCC 4.0 or later
